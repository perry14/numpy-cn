# 常用 API

本章按功能分组介绍了各常用的API。许多的API包含示例代码，这些示例代码演示了API的基本用法。

这些示例都是使用NumPy并且是通过这种方式导入NumPy：

``` python
>>> import numpy as np
```

::: tip 提示

执行示例的一种便捷的方法是 IPython 的 %doctest_mode 模式，它允许粘贴多行示例并保留缩进。

:::
